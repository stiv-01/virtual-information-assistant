# virtual-information-assistant
Trabajo de Integración Curricular - Diseño e implementación de un asistente virtual para brindar información de un Congreso / Jornadas Científicas.
Desarrollado en Godot Engine 4, con Ollama y Phi-3 mini con una arquitectura basada en Edge Computing.
